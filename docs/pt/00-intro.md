# Introdução

Bem-vindo ao **observability-sensei**.

Se você já passou por deploy com build verde e produção quebrada, este repo é para você.

## O que é

Projeto hands-on com:

- API TypeScript instrumentada para produção
- Pipelines GitHub Actions copiáveis
- Scripts bash (e PowerShell) para health check, métricas e rollback
- Stack Docker: Prometheus, Grafana, Jaeger

## Para quem é

- Devs backend entrando em CI/CD
- DevOps ensinando júniores
- Tech leads que querem ir além de "pipeline verde = pronto"

## Ideia central

> **CI diz se compila. Observabilidade diz se o release funciona.**
